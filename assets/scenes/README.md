# 场景图放置说明

把 AI 生成的场景图放到本目录，并使用以下文件名：

- `store-service-needs.jpg`
- `product-introduction.jpg`
- `inventory-allocation.jpg`
- `channel-weekly-review.jpg`
- `cross-cultural-alignment.jpg`

建议：

- 尺寸比例：`16:9` 或 `4:3`
- 最小宽度：`1200px`
- 文件格式：`.jpg`（也可改代码支持 `.png`）
- 风格：浅色背景、人物动作明确、留白适中

如果文件不存在，页面会自动回退到渐变背景，不影响功能。
